# StudyLog
学習習慣を可視化するために開発したパーソナルツールです。
必要最低限の機能を自作することで、学習の仕組み化を図っています。
学習継続のモチベーションを高めるため、UIデザインにもこだわって制作しました。
<img width="1470" height="827" alt="studylog" src="https://github.com/user-attachments/assets/fd3518e9-7d11-4c39-a73b-16f2f62925a9" />

## 主な機能
- 学習時間の計測と記録
- カテゴリ別の学習量可視化

## 使用技術
- Ruby on Rails
- HTML / CSS / JavaScript
- Docker
